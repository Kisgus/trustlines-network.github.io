Bootstrapping

While there is certainty on the technical feasibility of the system, there is still the bigger challenge of bootstrapping the network by attracting an initial user base. The first step towards global financial empowerment powered by Trustlines, is to build up a community of dedicated evangelists.
We believe the main ingredients for a dynamic community is openness, humbleness and a shared passion for global financial empowerment. A set of values we pledge our self and the community to enforce through social norms.
